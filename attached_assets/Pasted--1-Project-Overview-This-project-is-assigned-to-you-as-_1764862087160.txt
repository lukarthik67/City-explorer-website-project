✅ 1. Project Overview 
This project is assigned to you as part of your Frontend Internship Program at 
B2World.The purpose of this project is to assess your frontend development skills, 
including your ability to design user interfaces, handle structured data on the frontend, and 
build a fully responsive and interactive website. 
You are required to build a frontend-only blogging and city information website where 
users can explore different cities, read blog-style content, and view famous tourist places, 
food, and travel information. 
⚠️ Important: This is a pure frontend project. No backend, database, or authentication 
system is required. 
 
✅ 2. Tech Stack (Mandatory) 
You may use either of the following: 
 HTML5 
 CSS3 (Flexbox, Grid, Responsive Design) 
 JavaScript (ES6+) 
✅ Optional & Bonus: 
 React.js 
 Bootstrap / Tailwind CSS
